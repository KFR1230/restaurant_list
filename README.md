# restaurant_list

## Introduction 
This is my homework from Alpha camp.

## Prerequisites
- Node.js 14.16.0
- Express: "^4.18.2"
- Express-handlebars: "^3.0.0"
- Font-awesome: "^4.7.0"
- Nodemon 2.0.20
- Bootstrap 5.3.0

## Installation and Execution
1. 打開你的Terminal，進入存放此專案的資料夾
* cd rest_list *
2. 安裝npm套件
* 在Terminal 輸入 npm install 指令 *
3. 執行專案
* 在Terminal 輸入 npm run dev 指令 *
4. 若在終端機上看到此行代表順利運行，打開瀏覽器進入到以下網址
* http://localhost:3000 *
5. 若想暫停使用
* ctrl + c *

## Features
- 可以透過搜尋欄，快速找到特定的餐廳
- 點擊餐廳可以看到詳細資訊
